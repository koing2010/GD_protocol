# USB_Printer
usb virtual printer

## 1 MatchingIterm

### 1.1建立匹配项 A1+A2+~
		{"现" + "金"}
		{"现" + "价"} 等等
		
### 1.2匹配规则
		Iterm = { A + 数字字符开始 + 数字字符结尾}
	
	
## 2 Loading MatchingIterm
 
###	 2.1 format: 
		{{{现金}{现价}{总额}{合计}{总价}}}
###	  以两个左大括号 '{{'开始，两个右 '}}'  结束    
大括号字符不是常打印字符，容易检索
###	  目前先固定两个汉字，如果不设置的话，用四个英文空格(hex = 0x20)代替
###   设置完成验证
把设置的**Iterm**+**金额**分别按发送出去**蜻蜓**端出现收款提示为正确  
如依次发送： 
    **“现金 0.1 元”**  
	**“现价 0.2 元”** 等等
